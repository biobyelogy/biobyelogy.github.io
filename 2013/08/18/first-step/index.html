转行第一步

发表于 2013-08-18 | |
| 字数统计 115 字 | 阅读时长 1 分钟

寻找新行业是转行第一步。可以参考果壳网的各学科入门书单。不过这么多书一下子读不下来，也比较抽象。了解新行业一个好办法就是，找你的朋友向你介绍他们的行业：这个行业做什么，所需技能，典型的每日的工作流程，社会需求，发展空间。找 5、6 个朋友聊完，你就会有些眉目了。 ​​​​
