问答：生物大一怎么转行

发表于 2014-08-13 | |
| 字数统计 81 字 | 阅读时长 1 分钟

问

生物专业刚刚大一，想考研转行，考什么比较好？

答

什么意思？大学四年已经放弃，就指望研究生了？转行只要两年，本科转两次都够了。

方法：

  1. 刷高 GPA；
  2. 转系、辅修、双学位；
  3. 跨专业实习；
  4. self-project。

把握当下，大有可为。
